# V&C Quantum Hub 🚀

**V&C Quantum Hub** é um CRM de alta performance desenvolvido para consultores da V&C, focado na gestão ágil de vendas, comissões e bônus. O sistema foi projetado com uma interface futurista, rápida e focada em resultados.

## ✨ Funcionalidades Principais

- **📊 Dashboard Quantum:** Acompanhamento em tempo real de metas de produtos, assistência e impermeabilização.
- **💰 Gestão de Comissões:** Cálculo automático de comissões base (2.2%), bônus de serviços fixos e bônus acelerador por nível de performance.
- **🚫 Cancelamento Inteligente:** Sistema de cancelamento de pedidos que mantém o histórico mas remove automaticamente os valores dos cálculos financeiros.
- **📱 Relatórios para WhatsApp:** Geração de relatórios formatados para compartilhamento rápido com supervisores e equipes.
- **🌐 Offline-First:** Funciona perfeitamente sem internet, sincronizando dados localmente via `LocalStorage`.
- **🎨 Interface Futurista:** Design inspirado em interfaces de alta tecnologia, com animações suaves e tipografia marcante.

## 🛠️ Tecnologias Utilizadas

- **React 18+**
- **TypeScript**
- **Tailwind CSS** (Estilização Utilitária)
- **Lucide React** (Ícones)
- **Framer Motion** (Animações)
- **Vite** (Build System)

## 🚀 Como Rodar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/vc-quantum-hub.git
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd vc-quantum-hub
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Abra no navegador:**
   O projeto estará rodando em `http://localhost:3000`.

## 📂 Estrutura de Arquivos

- `src/App.tsx`: Componente principal e lógica de estado.
- `src/tipos.ts`: Definições de interfaces e enums do TypeScript.
- `src/components/`: Componentes reutilizáveis (Formulário, Sidebar).
- `src/constants.tsx`: Constantes e configurações do sistema.
- `src/index.css`: Configurações globais de estilo e Tailwind.
